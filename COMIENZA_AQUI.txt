PASO 1: OBTÉN API KEY (2 minutos)
───────────────────────────────────────────────────────────────────────────────

→ Ir a: https://console.anthropic.com/account/keys

→ Copiar una API key (ejemplo: sk-ant-bWVudGFsLWFjdGl2...)

→ GUARDAR EN LUGAR SEGURO


PASO 2: GITHUB (3 minutos)
───────────────────────────────────────────────────────────────────────────────

→ Crear cuenta en GitHub: https://github.com/signup

→ Crear repo nuevo en: https://github.com/new
   Nombre: usrpt-trainer
   Public
   ✓ Initialize with README

→ Clonar en tu PC:
   git clone https://github.com/TU_USUARIO/usrpt-trainer.git
   cd usrpt-trainer

→ Copiar TODOS los archivos que creamos en la carpeta

→ Crear estructura:
   usrpt-trainer/
   ├── src/
   │   ├── App.jsx         ← COPIA AQUÍ
   │   ├── App.css         ← COPIA AQUÍ
   │   └── main.jsx        ← COPIA AQUÍ
   ├── server.js           ← COPIA AQUÍ
   ├── package.json        ← COPIA AQUÍ
   ├── vite.config.js      ← COPIA AQUÍ
   └── ... (resto de archivos)

→ Hacer push:
   git add .
   git commit -m "USRPT Trainer App"
   git push -u origin main


PASO 3: VERCEL - FRONTEND (5 minutos)
───────────────────────────────────────────────────────────────────────────────

→ Ir a: https://vercel.com/signup
   Registrarse con GitHub

→ Click "New Project"
   Seleccionar: usrpt-trainer

→ Deploy (sin cambiar nada)

→ Esperar 1-2 minutos

→ URL aparece (ejemplo: https://usrpt-trainer.vercel.app)

→ GUARDAR URL


PASO 4: RAILWAY - BACKEND (5 minutos)
───────────────────────────────────────────────────────────────────────────────

→ Ir a: https://railway.app
   Sign up con GitHub

→ Click "New Project"
   "Deploy from GitHub"
   Seleccionar: usrpt-trainer

→ Esperar a que termine el deploy (~1 min)

→ En Railway → Variables
   Agregar dos:
   
   ANTHROPIC_API_KEY = sk-ant-tu_api_key
   NODE_ENV = production

→ Railway → Settings → Public Networking
   Copiar URL (ejemplo: https://usrpt-trainer-prod.up.railway.app)

→ Volver a Vercel → Settings → Environment Variables
   Crear/Editar VITE_API_URL:
   https://usrpt-trainer-prod.up.railway.app
   
   Redeploy automático (~30 segundos)
